# shared/ts

프로젝트 전반에서 재사용되는 핵심 TypeScript 로직 저장소입니다.
외부 의존성을 최소화하고, 안정적인 타입 정의와 순수 유틸리티를 제공하는 것을 목표로 합니다.
*이 패키지는 모든 프론트엔드 환경(React, Vanilla JS 등)에서 공통으로 참조할 수 있습니다.*
